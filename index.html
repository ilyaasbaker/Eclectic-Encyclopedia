Welcome to the Eclectic Encyclopedia ✱

The Eclectic Encyclopedia is a creative, self updating web database that has all types of factual data from which to learn. Each image links to a page of relavant factual information. For example; click on a beetle to learn about 1 of 400,000 species!

The content will refresh every 24 hours, giving you plenty of time to absorb and enjoy fresh knowledge.

Check out the PORTFOLIO for this project if you want a more in depth explanation.

DISCLAIMER

For the time being only a few datasets are being displayed, a more diverse pool of information will be added later on.

Some data sets may display data completely unrelated to the page e.g. On the horse page because you may get information on a "Horseshoe Crab" or "Horsefly"; because the search term is "Horse". So anything that contains the search term will be added to the pool of information that'll be displayed on that page.

This is a filtering issue which I'm working on fixing, so that the data accuracy is 100%.
